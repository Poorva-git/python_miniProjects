# 🐍 Python Mini Projects

A collection of beginner-friendly Python projects built to practice core programming concepts such as loops, functions, conditional statements, file handling, and basic GUI development.

These projects helped strengthen my problem-solving skills and understanding of Python fundamentals.

---

## 📌 Projects Included

| Project | Description |
|---------|-------------|
| 🧮 Calculator | Performs basic arithmetic operations like addition, subtraction, multiplication, and division. |
| 🎲 Number Guessing Game | Guess a randomly generated number with hints. |
| ✊ Rock Paper Scissors | Classic game against the computer using random choices. |
| 🏠 Rent Calculator | Calculates per-person rent including electricity and other expenses. |
| 🔐 Password Generator | Generates secure random passwords of user-defined length. |
| 📝 To-Do List | Add, view, and remove daily tasks using a menu-driven program. |
| 📞 Contact Book | Store, search, view, and delete contact details. |
| 💰 Expense Tracker | Track daily expenses and calculate total spending. |
| ⏰ Digital Clock | Live digital clock built using Python Tkinter. |
| ❓ Quiz Game | Simple multiple-choice quiz that calculates the final score. |

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Poorva-git/python_miniProjects.git
```

2. Open the project folder.

3. Run any project using Python.

```bash
python calculator.py
```

Replace `calculator.py` with the project you want to execute.
